INTRODUCTION                                    *todo-navigator*

The vim-todo-navigator plugin helps you quickly find and navigate TODO, 
FIXME, NOTE, and other custom comment tags in your codebase. It provides a 
convenient interface to search across multiple file types while respecting 
directory exclusions.

USAGE                                           *todo-navigator-usage*

Basic usage:
>
    :TodoNavigator
<
This opens a split window showing all TODO-style comments found in your 
project. Use <Enter> to jump to any item, and 'q' or <Esc> to close.

The plugin searches recursively from the directory where Vim was opened,
respecting configured file extensions and directory exclusions.

==============================================================================
COMMANDS                                        *todo-navigator-commands*

                                                *:TodoNavigator*
:TodoNavigator          Opens the TODO navigation window showing all found
                        comments matching the configured keywords.

                                                *:ShowTodos*
:ShowTodos              Alias for :TodoNavigator

                                                *:TODOToggle*
:TODOToggle             Toggles the TODO navigation window. If open, closes it.
                        If closed, opens it.

==============================================================================
CONFIGURATION                                   *todo-navigator-configuration*

                                                *g:todo_navigator_keywords*
g:todo_navigator_keywords
                        List of keywords to search for in comments.
                        Default: ['TODO', 'FIXME', 'NOTE', 'HACK', 'BUG', 
                                 'CANCELLED', 'XXX']
                        
                        Example: >
                        let g:todo_navigator_keywords = ['TODO', 'FIXME', 'REVIEW']
<

                                                *g:todo_navigator_file_extensions*
g:todo_navigator_file_extensions
                        List of file patterns to search within.
                        Default: ['*.py', '*.js', '*.ts', '*.java', '*.c', 
                                 '*.cpp', '*.h', '*.html', '*.css', '*.md']
                        
                        Example: >
                        let g:todo_navigator_file_extensions = ['*.py', '*.vim']
<

                                                *g:todo_navigator_exclude_dirs*
g:todo_navigator_exclude_dirs
                        List of directory names to exclude from search.
                        Default: ['.venv', 'venv', '__pycache__', '.git', 
                                 'node_modules']
                        
                        Example: >
                        let g:todo_navigator_exclude_dirs = ['.git', 'build', 'dist']
<

                                                *g:todo_navigator_initial_cwd*
g:todo_navigator_initial_cwd
                        Base directory for searches. Automatically set to the
                        directory where Vim was opened. Usually not modified
                        by users.

==============================================================================
MAPPINGS                                        *todo-navigator-mappings*

In the TODO navigation buffer:

<Enter>                 Jump to the selected TODO item
q                       Close the TODO navigation window  
<Esc>                   Close the TODO navigation window

Recommended global mapping: >
    nmap <F5> :TODOToggle<CR>
<

==============================================================================
EXAMPLES                                        *todo-navigator-examples*

Basic configuration in your .vimrc: >
    " Set custom keywords
    let g:todo_navigator_keywords = ['TODO', 'FIXME', 'NOTE', 'REVIEW']
    
    " Search only in Python and JavaScript files
    let g:todo_navigator_file_extensions = ['*.py', '*.js']
    
    " Add more directories to exclude
    let g:todo_navigator_exclude_dirs = ['.venv', '.git', 'node_modules', 'build']
    
    " Set up toggle mapping
    nmap <F5> :TODOToggle<CR>
<

Example output window: >

COLOR HIGHLIGHTING                             *todo-navigator-colors*

Different keywords are highlighted with different colors:

TODO        Green       (tasks to be done)
FIXME       Red         (things that need fixing)  
NOTE        Cyan        (important notes)
HACK        Orange      (temporary workarounds)
BUG         Red         (known issues)
CANCELLED   Gray        (cancelled items, with strikethrough)
XXX         Red         (urgent attention needed)

Custom keywords get basic syntax highlighting. You can define custom colors
by adding highlight commands in your .vimrc: >
    highlight TodoKeyword_REVIEW ctermfg=blue guifg=blue
<

==============================================================================
TROUBLESHOOTING                                 *todo-navigator-troubleshooting*

No results found:
- Check that your files contain the keywords you're searching for
- Verify file extensions match your |g:todo_navigator_file_extensions|
- Ensure directories aren't excluded by |g:todo_navigator_exclude_dirs|
- Try running the plugin from the project root directory

Performance issues:
- Add large directories to |g:todo_navigator_exclude_dirs|
- Limit |g:todo_navigator_file_extensions| to only needed file types
- Consider the size of your project directory
